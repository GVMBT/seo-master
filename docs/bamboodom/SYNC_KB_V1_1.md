## Что в новом разделе 1.5

Универсальные правила монтажа декоративных стеновых панелей. 7 подразделов:

1. **1.5.1. Требования к основанию** — 5 типов поверхностей (штукатурка, ГКЛ, монолит, газобетон, плитка) с конкретными требованиями для каждой + общие запреты (мокрая штукатурка, меловая краска, остатки обойного клея).

## Источник расширения

Расширение взято из **отфильтрованного отраслевого гайда по монтажу декоративных панелей**. Оригинал описывал ПВХ-панели, не WPC. Применены только универсальные части:

**Взято (универсальное):** подготовка основания, инструмент, общие ошибки, базовая технология монтажа, чек-лист, FAQ про детали монтажа.

**НЕ взято (противоречило продукции Дизайн-Сервиса):**
- Раздел про «ПВХ-сердечник» — у WPC композитная структура из бамбукового волокна с полимером, не ПВХ
- Раздел про фрезеровку углов с V-бороздой и прогревом строительным феном — у Дизайн-Сервиса метод через алюминиевые XHS-профили (проще, применим на всех материалах)
- Раздел про гибкий камень как срез песчаника — это другой материал, не flex Дизайн-Сервиса (flex — композит на сетке, имитация камня, не натуральный)

## Что это даст для итерации 4B.2

**Проблема 4B.1** (про которую я писал в `FEEDBACK_ITERATION_4B_1.md`): AI галлюцинирует цифры и артикулы чтобы добрать до 1500 слов, потому что KB недостаточен для глубоких тем типа «WPC в спальне» или «монтаж в коридоре».

**Теперь в KB есть фактура для:**

- **Подготовки основания** — раньше AI писал общее «стена должна быть сухой», теперь может разобрать 5 типов оснований с конкретикой
- **Инструмента** — раньше не упоминал, теперь полный список
- **Ошибок монтажа** — 7 конкретных ошибок с объяснением последствий
- **Частных вопросов клиентов** — можно использовать для формата 6 (Q&A) и для боковых разделов формата 1 (гайд-инструкция) и формата 3 (use-case)

### Рекомендация

При обновлении промпта до **v7** (после v6 с RULE 4):

```
При нехватке слов до 1500 — расширяй секцию через:
1. Детализацию подготовки основания (KB раздел 1.5.1) —
   разбери 2-3 типа оснований применительно к теме
2. Список ошибок монтажа (KB раздел 1.5.6) —
   3-5 ошибок с объяснением последствий
3. Q&A секция с практическими вопросами (KB раздел 3 «Про детали монтажа»)

НЕ добирай через галлюцинации цифр (RULE 4).
Если после использования этих источников всё ещё < 1500 — вернуть
ошибку "insufficient KB for topic".
```
